Config

Let's show you how to take full advantage of _Project_ by setting the various config options available to you.

Useful for: Customizing Behavior, Automating More Tasks, Disabling Unwanted Features

Passing In Config Options

First, lets show you how to pass in a config object. Then we'll show you what's available.

Pass In A Config Object

Param
config Object

You will start by making a new config object, then pass it in on generation.

jQuery(function($) {
  
  window.plugin = $('#element').plugin({ random: true, transition: 'fade', interval: 10000 });
  
  // or be easier to read
  
  var config = {
  	random: true,
  	transition: 'fade',
  	interval: 10000 // 10 seconds
  };
  
  window.plugin = $('#element').rotator(config);
  
});

Available Config Options

Here are all of the config options available to you upon generation of your rotator.

config.id

Param
id Int

This is the id of the element. Unique integer.

config.name

Param
name String
Default Value
'default'

This is the name of the element. The name is pulled from the attribute you set in config.id.

Note: The name does not have to be unique, it is for informational purposes only.

config.description

Param
description String
Default Value
''

This is the description of the element.

config.location

Param
location String top | right | bottom | left
Default Value
'top'

This is the location of the element. Output will vary depending on location.

Building Your Own Transitions

Adding your own transition is easy to do once you know the basics. Read on below to start building your own transitions.

base.next()

Returns
Object (Base)

Changes the item to the next in the list. If no next item exists, we don't change.

// go to next item
base.next();

// or go a few later
base.next().next();

base.prev()

Returns
Object (Base)

Changes the item to the previous in the list. If no previous item exists, we don't change.

// go to previous item
base.prev();

base.goTo(id)

Param
id Int
Returns
Object (Base)

Go to a specific item based on it's id.

// go to specific item
base.goTo(10);

base.reorder(by, sort, refresh)

Param
by String
Param
sort String asc | desc
Param
refresh Bool
Returns
Object (Base)

Reorder the items in the list. by can be set to any of the keys held in the item. sort is the direction in which to sort the items. And finally refresh will rewrite the items to the DOM so the new order is reflected in the page.

// reorder the list based on date, descending and then write it to the DOM
base.reorder('date', 'desc', true);
