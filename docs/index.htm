Overview

Learn what Docs is all about and how you can use it.

What does it do?

Docs is a simple one-page HTML template for documentation of a project. It is made to provide documentation for smaller projects that don't need a whole user guide. An ideal use of Docs would be for a project with 5-6 basic sections that need to be covered.

What are some good uses for Docs?

Docs is perfect for your JavaScript plugin, basic framework, module, or small API.

How hard is it to implement?

There's only a few main steps to take to get your rotator up and running.

Copy and rename the _template folder in your project:

  • Copy the _template folder from the root of the repo
  • Paste it into your project and rename it to docs or whatever you'd like

Replace the following variables in the index.htm file:

  • _Project_ becomes your project name, e.g. Docs or Docs API
  • _Project Description Short_ becomes your project short description (one sentence), e.g. A one-page documentation template; used in page title
  • _Project Description_ becomes your project description, e.g. A one-page documentation template built to help you quickly set up documentation for your project
  • _Company_ becomes the company or person who holds the license for this project, e.g. cmfolio
  • _Company URL_ becomes your company or personal url, e.g. http://web.cmfolio.com

How can I customize Docs?

Since you already have full control over the HTML you generate, we provide some basic themes for you to start with. You can either take out themes and modify theme or create a new one from scratch.
